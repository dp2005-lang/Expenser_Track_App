# Expenser_Track_App
# 💰 Expense Tracker App (Google Colab Version)

## 📌 Project Overview

The **Expense Tracker App** is a data science project designed to help users track, analyze, and visualize their financial transactions. This version is built using **Google Colab**, making it easy to run without any local setup.

The project uses **synthetic data** to simulate real-world financial transactions and demonstrates how data analysis can be applied in **FinTech applications**.

---

## 🎯 Objective

* Track daily expenses and income
* Categorize spending (Food, Travel, Rent, etc.)
* Analyze financial behavior
* Generate insights for better decision-making

---

## 🧠 Problem Statement

Managing personal finances manually is difficult and error-prone. Raw transaction data (like bank statements) is hard to interpret.

This project solves that problem by:

* Structuring expense data
* Analyzing trends
* Visualizing spending patterns

---

## 🚀 Features

* ✅ Synthetic dataset generation
* ✅ Data cleaning & preprocessing
* ✅ Category-wise expense analysis
* ✅ Monthly spending trends
* ✅ Data visualization (bar, pie, line charts)
